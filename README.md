Airline-Booking-Dashboard

Overview
This interactive dashboard provides real-time insights into airline booking patterns, customer preferences, and sales performance across multiple channels and routes.
Key Features
Core Metrics

Total Passengers: Track overall passenger volume (79,562 passengers)
Average Flight Duration: Monitor flight time metrics (7.28 hours avg)
Baggage Preferences: Analyze extra baggage requests (66.88%)
Seat Preferences: Track preferred seat selections (29.70%)
In-flight Meals: Monitor meal service requests (42.71%)

Booking Analytics

Daily Booking Completion: Visual bar chart showing booking trends across the week
Peak Performance Days: Identify high-volume booking periods
Weekly Pattern Analysis: Compare booking patterns day-by-day

Route Intelligence

Top 10 Most Requested Routes:

AKLKUL (4,450 bookings)
PENTPE (1,566 bookings)
DMKKIX (1,428 bookings)
And more...


Geographic distribution map showing passenger origins worldwide

Sales Performance

Multi-Channel Tracking:

Internet sales (5,618)
Mobile sales (44,382)


Channel Comparison: Donut chart visualization for sales distribution

Smart Filtering

Sales Channel Filter: Internet, Mobile
Trip Type Filter: Circle Trip, One Way, Round Trip
Flight Day Filter: Sunday through Saturday selection

Technologies Used

Visualization Tool: Microsoft Excel
Charts: Excel Charts & Pivot Tables
Mapping: Excel 3D Map / Bing Maps
Design: Custom color scheme with teal/blue gradient theme
Data Analysis: Excel Formulas & Functions

Getting Started
Prerequisites

Microsoft Excel 2016 or later (for best compatibility)
Excel features required: Charts, Pivot Tables, Slicers, 3D Maps

Data Sources
The dashboard analyzes:

Booking completion records
Passenger preference data
Route information
Sales channel metrics
Geographic booking origins

Dashboard Sections

Summary Cards - Quick overview of key metrics
Temporal Analysis - Booking completion by day
Route Performance - Most popular flight routes
Sales Analytics - Channel performance breakdown
Geographic View - Global passenger distribution map


Usage Tips

Use the sales_channel slicer to compare Internet vs Mobile performance
Filter by trip_type to analyze different booking patterns
Select specific flight_day to see daily trends
Hover over charts for detailed data points
Right-click on pivot tables to refresh data
Use Excel's filter buttons to drill down into specific metrics
